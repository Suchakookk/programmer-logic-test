# Programmer Logic Test (SQL Server / HTML-CSS / C#)

- SQL Server
- HTML & CSS
- C#

## Project Structure
- `docs/` : ไฟล์โจทย์ต้นฉบับ
- `sql/` : SQL (schema, seed, answers)
- `html-css/` : HTML/CSS
- `csharp/HelloWorld/` : C# (Console)

## How to Run

### SQL Server
1) เปิด SQL Server Management Studio (SSMS)
2) รันไฟล์ตามลำดับ:
- `sql/01_schema.sql`
- `sql/02_seed.sql`
- `sql/03_answers.sql`

### HTML & CSS
- `html-css/01_layout.html`
- `html-css/02_table.html`

### C#
```bash
cd csharp/HelloWorld
dotnet run
```
